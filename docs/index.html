@magic-themes/docs

this is the @magic-themes docs theme. It is being used as the documentation theme for most @magic packages.

installation:

installation is done using npm. for now, all magic themes are living on github and not on npm.

note the missing @ before magic-themes.

this is how we install npm modules from github.

npm install magic-themes/docs

require in config.js:

// config.jsmodule.exports = {  // ... other config  THEME: 'docs',}

magic will then find the theme automatically, just as it did on this page
